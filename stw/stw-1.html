Getting Started

Goals for this lesson are
  1. Download the necessary software
  2. Create your first webpage
  3. Set up and publish a GitHub pages website

1. Download the necessary software

First, we need to get some software to help make this easier. In genergal, you do not need any of this. If you wanted, you could just use a simple text editor such as notepad to make your website. However, for the purpose of a tutorial it is much easier if you are using the same software that I am.

It is not extremely important, but for your reference I am using an m2 Macbook Air running macOS v.14. Everything here should work fine on Windows, and if you are running Linux then figure it out yourself!

Here is our shopping list: (everything is free)

  • Microsoft Visual Studio Code (VS Code)
  • Firefox web browser
  • GitHub Desktop

Basically, we will write our code in VS Code and view our work in Firefox. To publish our website we will use a GitHub sevice called GitHub Pages which will use GitHub Desktop. If you don't have a GitHub account you can create one now. Be sure to choose a good username because your website's url will be username.github.io

Before we finish here we need to install an extension in VS Code. Press Command + Shift + X to open the extensions menu. Search for the extension called "Live Preview" and install it. You can also find it here.

This extension will allow us to view our website locally on our computer in the browser and see our changes updated in real time. Cool!


2. My first website

Now that you have everything installed lets make a (very) simple website!

Lets go to our documents folder and create a new folder called "webTest".

Next, open VS Code and click File > Open Folder. Find your webTest folder and open it. On your left is the "EXPLORER" window. Near the top you can find a button called New File... Click this and create a new file called index.html.

Now write the following into your file. 

                
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My first website!</title>
</head>
<body>
<p> My first website! </p>
</body>
</html>

This is a what a website looks like in HTML. Let's view this new page in the browser! Click on the preview button near the top right of the window. This will open your website in a small window next to your code. If you want to see it in another window you can copy and paste the URL from the preview into Firefox.

Try erasing the phrase "My first website!" and typing something else between the <p> and </p>. As you type, you will notice your changes in your Firefox window (or the preview if you still have it open). Very cool!

This is only a preview of your website. It only exists temporarily and is not "published."


3. Set up GitHub pages

To publish a website you would need to purchase the rights to a domain name, then host your site there. However, GitHub allows us to host one website for free! Let's set one up now.

Follow this guide: https://pages.github.com/

After step 1 you should select the option to view the tutorial using GitHub Desktop (we downloaded this earlier!)

After you create your new repository, you can create a new index.html file like the guide asks, or you can copy and paste the index.html file we created before. It is up to you! We will completely start over in the next step in this guide.

Once you commit and push your changes, you should try to visit your new website in you browser.


Congratulations!

You are now the owner of a working website!
