About this website

I created this site in Jan. 2024 for three purposes: to share my professional work, to learn how to make a website, and for fun.

Originally, I only wanted to use very basic HTML to give the website a cool early internet theme. However, learning web design has been a lot more fun than I thought, so I have enjoyed learning some more "modern" CSS while still trying to stay true to the simple theme. I also find Javascript to be a nice way to make visual results for problems in mathematics which are easy to share over the internet.

I still have a lot to learn so if you have any comments or suggestions I would love to know! Please send me an email at Justin [dot] Swain [at] unt [dot] edu.
