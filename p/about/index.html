About this website

I created this site in Jan. 2024 for two purposes: to post my professional work, and to learn HTML and CSS.

Originally, I only wanted to use very basic HTML to give the website a cool 90s theme. However, learning web design has been a lot more fun than I thought, so I have enjoyed learning some more "modern" CSS while still trying to stay true to the "simple" theme.

I still have a lot to learn (like js) so if you have any comments or suggestions I would love to know! Please send me an email at Justin [dot] Swain [at] unt [dot] edu.
