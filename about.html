About this website

Created in January 2024. My goal for this website was to use only HTML and inline CSS to acheive a "simple" early internet theme. However, spending several hours working on HTML then opening the page on an iPhone and seeing a huge mess was ... annoying.

Therefore, I did have to learn (and am still learning) some more modern website design but I still try to stay true to the original design idea. If you have any comments or suggestions I would love to know! Please send me an email at Justin [dot] Swain [at] unt [dot] edu.
